### 加速原理分析

**时间维度：** 理论最优加速比 = 分组数（5），实际 2.56x。差距来自：
- API 服务器端并发连接数限制
- 各分组气泡数量不均匀（1-4个），单分组耗时差异大
- 网络延迟和服务器端排队

**内容维度：** 并行模式每个分组独立调用 AI，不受其他分组干扰，每个分组都能充分利用完整的输出 token 预算，总输出量更大、内容更丰富。
