## 三、质量差异根因分析

**为什么并行模式内容更多（+73.6%）？**

串行模式把所有分组一次性发送给 AI，单次输出的总 token 预算受限。各分组之间存在竞争关系：
- 分组 1 输出多了，分组 2-5 必须压缩
- AI 会均衡各分组篇幅，导致每个分组都不够深入

并行模式每个分组独立调用 AI：
- 每个分组独占完整的输出 token 预算
- 不受其他分组干扰，可以充分展开
- 整体信息量大幅增加

**为什么并行模式速度更快（2.20x）？**

理论最优加速比 = 分组数（5），实际 2.20x。原因是：
- API 服务器端并发限制（DeepSeek 对单连接限速）
- 各分组气泡数量不均，最长分组决定总时间
- 网络排队延迟
